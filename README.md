# Tic-Tac-Toe #

**To Start**
```
npm install
bower install
npm start
```

`http://localhost:8003`

*note*: In order for dev building to work Ruby and the sass gem need to be install.

### Libraries Used
* AngularJS
* Jade - templating
* SASS - for all css
* Jasmine - Testing
* Karma - Test running
* Grunt - for building jade files/sass
* Node - for running dev server
* Bower - installing frontend dependencies
